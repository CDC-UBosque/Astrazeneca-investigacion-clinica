2.3. Tablas y Gráficos Para Describir Variables - Sección I

La tabla es quizás el método más sencillo para describir datos, una de las más utilizada es la distribución de frecuencias, si se trata de variables categóricas la tabla consiste en un conjunto de categorías con las cantidades correspondientes a cada una de ellas colocadas al frente, si se colocan las cantidades se habla de frecuencia absoluta, con el fin de permitir comparaciones con otras tablas se colocan los porcentajes a la derecha de las frecuencias absolutas, estas son las frecuencias relativas, si se trata de variables cuantitativas, se construyen las categorías en una serie de intervalos distintos que no se traslapen teniendo en cuenta que si hay pocas categorías se pierde información y si son muchas entonces no se justifica construir la tabla.

Es importante conocer una tabla ya que casi toda la información que entregan los paquetes de manejo estadístico de datos lo hacen en tablas. Las tablas son informativas cuando no son excesivamente complejas, debe encontrarse claramente el título de ella y el nombre de cada columna. Un tipo especial de tabla es la llamada tabla de contingencia de 2 x 2, de Fisher o tetracórica, que permite medir y evaluar la relación entre dos variables cualitativas dicotómicas, este tema es la base para la investigación analítica o correlacional.

Las gráficas representan datos numéricos a través de figuras, comunican los patrones generales de unas observaciones de una sola mirada, son más fáciles de leer que las tablas pero ofrecen menos información; el título, los nombres de los ejes y las unidades de medición deben indicarse con claridad. Los más usados son:

Diagramas de barras:

Se usan para variable nominales u ordinales, las diferentes categorías se presentan a lo largo del eje horizontal, se hace una barra separada (de la misma amplitud que las demás), sobre cada categoría de forma que cada barra represente la frecuencia (relativa o acumulada) de esa categoría, es más útil si se desea comparar categorías entre sí:

Histograma:

Es la gráfica más común, sirve tanto para variables discretas como continuas, el eje horizontal representa los límites exactos de los intervalos y el vertical la frecuencia absoluta o relativa en cada intervalo. Al hacer un histograma lo primero es decidir la escala de cada uno de los ejes, el vertical debe iniciar en cero para no distorsionar las comparaciones (ojo cuando encuentre ejes verticales cortados con dos líneas), el área de la barra es la frecuencia asociada con cada intervalo (la proporción del área total que corresponde a cada intervalo es la misma frecuencia relativa de cada intervalo).
