Media aritmética (promedio):

Medida más utilizada, se calcula sumando el valor de cada una de las observaciones y el total se divide entre el número de mediciones, se denota como una equis x con una barra en la parte superior:

La letra griega sigma (∑) es el símbolo de suma, las expresiones i=1 y n debajo y sobre sigma significan que se debe sumar los valores de las observaciones del grupo desde x=1 hasta x= n. La media sirve para variables continuas y discretas, pero no es para nominales u ordinales (no tiene lógica decir que el promedio del sexo es 1.4) Excepto cuando las variables son binarias y los posibles valores se representan con 1(presencia) o 0 (ausencia), en este caso la media de las observaciones es igual a la proporción de números uno (1) en el conjunto de datos (importante después en distribuciones binomial y de Poisson). Como el método para calcularla toma en cuenta la magnitud de cada observación los valores extremos cambian el resultado, dado que a veces se requiere describir todo un grupo de datos, en el caso de valores extremos se usan medidas de resumen central que no sean tan sensibles a valores extremos.

Mediana:

No es sensible a los valores extremos, sirve para variables discretas, continuas y ordinales, es el valor que se encuentra exactamente en la mitad de los datos ordenados (percentil 50) por ello, si las observaciones son impares la mediana es el valor que está en el medio de la lista ordenada, si es par se toma como el promedio de los dos valores centrales.
